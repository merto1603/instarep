# instareport
Instagram kullanıcı hesabını raporlama


İndir [Termux](https://play.google.com/store/apps/details?id=com.termux)

```bash
pkg install git
pkg install python
git clone https://github.com/merto1603/instarep.git
cd instarep
python3 -m pip install requests
python2 termux.py
```
